```
backend/

в”Ӯ   вҖў Scrapes 3 pages

в”Ӯ   вҖў Returns 30 houses
в”Ӯ

в”Ӯ   вҖў Run once to get data
в”Ӯ   вҖў Saves to cache
в”Ӯ   вҖў Shows sample output
в”Ӯ

в”Ӯ   вҖў GET /housing/coimbatore
в”Ӯ   вҖў Returns cached/fresh data
в”Ӯ
в”ңв”Җв”Җ .env                        вҶҗ API key
в”Ӯ   вҖў RAPIDAPI_KEY=...
в”Ӯ
в””в”Җв”Җ coimbatore_houses_cache.json вҶҗ Cached data
    вҖў 30+ houses
    вҖў Photos, descriptions
    вҖў Auto-generated

src/services/
в””в”Җв”Җ datasetsApi.ts              вҶҗ Frontend API calls
    вҖў loadHouses() function
    вҖў Calls backend endpoint
    вҖў Handles errors
```
